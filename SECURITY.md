# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please follow the steps below to report it:

1. **Do Not Open an Issue**: Please do NOT open a public issue in the GitHub repository.
2. **Email the Security Team**: Instead, email security@yourdomain.com (replace with the actual email address) with the details of the vulnerability, including:
   - A description of the vulnerability
   - Steps to reproduce the issue
   - Any relevant screenshots or logs
3. **Wait for a Response**: We will respond to your report within 72 hours and work with you to resolve the issue.
4. **Disclosure Timeline**: Once the issue is fixed, we will discuss disclosure options with you, respecting your wishes and the severity of the vulnerability.

## Acknowledgments

We will thank you publicly in our repository if you would like, but please let us know your preference when reporting the issue.
